# ğŸ GPlayerBee

**GPlayerBee** is a client-side Minecraft mod focused on player visuals and lightweight client utilities.

The mod enhances player presentation with glow effects and cosmetic stat displays, while also providing optional quality-of-life features for a smoother client experience.

> Looks clean. Plays clean.

---

## âœ¨ Features

* **Player Glow Effect**
  Adds a clean glow effect to players for better visibility.

* **Friend System**

  * Add players to your friends list
  * Friends are highlighted with a **green glow**
  * Other players use the default glow color

* **Auto Attack (Optional)**
  A client-side auto attack feature.
  Auto attack will **never target friends**.

* **Cosmetic Statistics Display**
  Displays enhanced-looking statistics purely for visual purposes.

* **Attack Interaction Improvements**
  Allows attacks to pass through grass, cobwebs, and similar blocks.

* **Fully Client-Side**
  No server installation required. All features are configurable.
